# UI Customizations

You can customize some parts of the UI as showed below.

## Branding configuration (Logo)

You can customize the header and sidebar brand logos via config/env vars. Defaults keep the current UI unchanged.

**Env vars**

* `LOGO_HEADER_URL` — image URL (absolute or `/static/...`).
  *Default:* `/static/assets/img/hackinsdn.png`
* `LOGO_HEADER_ALT` — alt text / visible brand text.
  *Default:* `HackInSDN`
* `LOGO_HEADER_HREF` — link target when clicking the brand.
  *Default:* `/`
* `LOGO_SIDEBAR_URL` — image URL (absolute or `/static/...`).
  *Default:* `/static/assets/img/hackersdobem.png`
* `LOGO_SIDEBAR_ALT` — alt text / visible brand text.
  *Default:* `HackInSDN`
